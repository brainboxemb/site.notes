# CAD Software

This section contains notes and references for CAD software used for mechanical design.

## Software

- [OpenSCAD](/software/cad/openscad)  
  Script-based parametric CAD using the OpenSCAD language.

## Alternatives to Investigate

- [build123d](https://build123d.readthedocs.io/)  
  Python-based parametric CAD that may be worth investigating as an alternative to OpenSCAD.
